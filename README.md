## Quick Start

1. Clone this project
```bash

```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Configure your `.env` file:
   ```bash
   cp .env.example .env
   # Edit .env file with your API key
   vim .env
   ```

4. Run and start analyzing:
   ```bash
   npm run build
   npm run start
   ```
